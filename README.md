# Emil — 邮箱运维总控技能

Marvis 双向邮箱控制技能。支持弹窗邮件通知、进度报告推送、邮箱指令接收与执行。

## 功能
- 进度报告生成与邮件推送
- 邮箱指令队列处理（收邮件→执行→回复）
- 邮件自动发送（含附件，自动两阶段确认）
- 任务状态注册与更新
- 系统健康检查

## 安装
```
npx skills add LTX888888/emil
```
